◆ AUTOMATIC_TUNING

bool AUTOMATIC_TUNING = true

This flag sets whether the ground station should tune itself from the Function Id "10" transmissions. only used for prototyping, if the tuning system fails to allow hardware debugging.

Should use the Transceiver settings message to tune its own antenna.
