1. Setting Up

1. 1. Connecting to the Ground Station

Using the Ground Station's USB cable, plug it into a PC or Laptop (windows 7/10). On the Ground Station it will indiciate with [INSERT ICON HERE] that it is connected via USB.

1. 2. Starting the Ground Station software

Inside the files given with the Ground Station kit, there is a program called "Ground Station Relay Program". Run this executable and then open the control panel website.
